# AUA INCOME

Minimalist personal income tracker with Apple-inspired design.

## Features

- **Premium Design**: Apple-inspired minimalist interface with high-impact animations.
- **Neural Finance v2.0**: Advanced tracking of income and expenses.
- **Real-time Sync**: Powered by Firebase for instant data updates across devices.
- **Secure Authentication**: Google Login integration.
- **Responsive Activity**: Intelligent handling of large numbers and mobile-first navigation.
- **Custom Branding**: Fully integrated custom logo and identity.

## Tech Stack

- **Frontend**: React 18, Vite, TypeScript.
- **Styling**: Tailwind CSS, Framer Motion (motion/react), Lucide Icons.
- **Backend**: Firebase (Authentication & Firestore).
- **UI Components**: Radix UI / shadcn/ui.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Firebase project

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd aua-income
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env` file in the root directory and add your Firebase configuration (see `.env.example`).

4. Start the development server:
   ```bash
   npm run dev
   ```

## Deployment to Vercel

1. Push your code to a private GitHub repository.
2. Connect your repository to Vercel.
3. Add your environment variables in the Vercel project settings.
4. Deploy!
